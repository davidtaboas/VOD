## Extensions

- redux: storage of data
- Ducks Redux: redux architecture
- Isomorphic fetch: fetch data cross-browser
- Redux thunk: middleware for async actions
- Eslint aribnb: linting code
- react-icons: icons
- react-router: page routing
- react-moment: formating dates
- normalizr: structured data
- prop-types: validate props on react components
- lint-staged: validate, fix and format code on stage files

## Launch project

Only two commands:

```
npm install
```

and then

```
npm start
```

Note: I used node 6.10 and npm 3.10.10
